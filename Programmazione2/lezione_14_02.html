  • L'ambiente "IDLE" di Python non è un ambiente di sviluppo ma uno strumento per provare la singola riga di codice;
  • Python è un linguaggio "Case Sensitive" ossia riconosce la differenza tra lettere maiuscole e minuscole;
  • Infine IDLE usa i colori per distinguere i vari blocchi di codice;
  • Tuttavia, VS Code è l'opzione migliore a patto che si installino le relative estensioni a tema Python;
  • RICORDA: le versioni di Python sono in continuo cambiamento quindi ogni sua versione ha un inizio ed una fine;

  • Variabili e tipi di dati
  • Variabile: contenitore nominato di dati;
  • Nel linguaggio di Python si usa l'operatore di assegnazione = (segno uguale)
  • Rispetto ad altri linguaggi, Python ha una sintassi molto semplice ed intuitiva;
  • Una variabile ha un identificatore (cioè il suo nome) ed il simbolo uguale. Esempi:
    1. un_numero = 5
    2. una_stringa = "Ciao"
    3. lista_interi = [1, 2, 3]
  • Gli identificatori/nomi delle variabili devono avere delle regole:
    1. Non devono avere parole riservate come "if", "for", "return", ecc.
    2. Devono necessariamente iniziare con una lettera o un carattere di underscore(_)
    3. Possono contenere soltanto sequenze di lettere (minuscole e/o maiuscolte, esclusi i caratteri accentati), numeri e caratteri underscore(_)
    4. Anche gli identificatori sono "case sensitive" ossia distinguono tra lettere maiuscole e minuscole;
  • I tipi di dati supportati, di default, da Python si dicono "tipi built-in"
  • Tipi di dati numerici:
    1. Intero -> int
    2. Razionale (ossia con la virgola) -> float
    3. Complesso -> complex
  • Variabili, funzioni e metodi sono cose diverse
  • RICORDA: lo strumento "input" restituirà SEMPRE una stringa;

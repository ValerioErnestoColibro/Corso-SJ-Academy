Programmazione 2 - Lezione 7/02

  • Tratteremo Python, linguaggio molto usato soprattutto per l'IA;
  • Molto usato anche per la sua sintassi estremamente facile;
  • RICORDA: l'IA non sempre restituisce un giusto risultato perchè spesso NON SI FANNO LE DOMANDE GIUSTE;
  • Meglio scaricare la versione 3.11.5 e non le ultime versioni per incompatibilità tra le varie librerie;
  • Python è un linguaggio interpretato (il codice viene letto ed eseguito riga per riga da un interprete durante l'esecuzione. Questo permette maggiore flessibilità, ma può essere più lento.) e non compilato (il codice sorgente viene tradotto in un file eseguibile tramite un compilatore prima dell'esecuzione. Questo rende il programma più veloce, ma ogni modifica richiede una nuova compilazione.)
  • Per la sua versatilità, possiamo usarlo sia per script semplici che per programmi più complessi in cui subentra la programmazione ad oggetti;
  • Conviene usare IDLE Python come ambiente di programmazione INIZIALE;
  • Valerio, dai un'occhiata a ChatGPT;
  • Per installare le librerie di Python chiamate "numpy" e "matplotlib", digitare sul prompt Windows la scritta "pip install nomelibreria";
  • Per aggiornare le librerie, digitare sul prompt di Windows la scritta "python -m pip install --upgrade pip";

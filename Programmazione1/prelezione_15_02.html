Programmazione 1 - Prelezione 15/02

  • Java
  • Java è un linguaggio di alto livello, multipiattaforma e orientato agli oggetti;
  • E' indipendente dalla piattaforma grazie alla JVM;
  • Ha una grande comunità e molte librerie;
  • E' molto usato in aziende per applicazioni web, mobile e desktop;
  • Inizia installando la versione più recente di JDK da Oracle;
  • Controlla la tua versione digitando sul prompt dei comandi java -version
  • Se vuoi usare VS Code come editor, scarica un'estensione apposita;
  • Per il tuo primo programma in Java, crea un file chiamato HelloWorld.java
  • Poi scrivi questo codice:

    public class HelloWorld {

    public static void main(String[] args) {

    System.out.println("Ciao, mondo!");

    }

    }

Sistemi Operativi - Lezione 21/01

  • Git Bash è un'ottima interfaccia a riga di comando per lavorare con Git su Windows. Ecco alcuni dei comandi di base più utili:
    1. git config --global user.name "Nome Cognome" --> Imposta il nome dell'utente.
    2. git config --global user.email "email@example.com" --> Imposta l'email dell'utente.

  • Comandi di Inizializzazione e Clonazione
    1. git init --> Inizializza un nuovo repository Git.
    2. git clone (segnominore)repository_url(segnomaggiore) --> Clona un repository esistente.

  • Comandi di Stato e Log
    1. git status --> Mostra lo stato dei file nel repository.
    2. git log --> Visualizza la cronologia dei commit.

  • Comandi di Aggiunta e Commit
    1. git add (segnominore)file(segnomaggiore) --> Aggiunge file all'area di staging.
    2. git commit -m "messaggio del commit" --> Effettua un commit con un messaggio.

  • Comandi di Pull e Push
    1. git pull origin branch --> Estrae i cambiamenti dal repository remoto.
    2. git push origin branch --> Carica i cambiamenti nel repository remoto.

  • Comandi di Branching
    1. git branch --> Elenca i branch esistenti.
    2. git checkout -b (segnominore)nome_branch(segnomaggiore) --> Crea e passa a un nuovo branch.
    3. git merge (segnominore)nome_branch(segnomaggiore) --> Unisce un branch nel branch corrente.

  • Comandi di Reset e Revert
    1. git reset (segnominore)commit(segnomaggiore) --> Riporta l'HEAD a un commit precedente.
    2. git revert (segnominore)commit(segnomaggiore) --> Crea un nuovo commit che annulla le modifiche di un commit precedente.

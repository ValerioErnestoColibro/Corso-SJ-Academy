PHP - Lezione 5/02

  • La parte HTML di un progetto PHP è slegata dal resto;
  • Se si fa una chiamata al server, bisogna aprirla E POI CHIUDERLA SEMPRE per evitare la fuga di dati;
  • Le azioni di PHP sono principalmente 4:
    1. Inserisci dati;
    2. Ordina dati;
    3. Aggiorna dati;
    4. Cancella dati;
  • In PHP, conviene fare un form HTML per OGNI CHIAMATA;
  • IMPORTANTE: è normale non capire il codice all'istante;

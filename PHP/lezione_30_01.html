PHP - Lezione 30/01

  • Scaricare XAMPP;
  • Scaricare l'estensione PHP Elefantino su VS Code;
  • PHP ci permette di avere un sito web a basso costo (tipo 20 euro l'anno);
  • PHP è la base di strumenti come WordPress e Shopify;
  • PHP ci permette di costruire un buon lato back-end, gira sul server e non viene vista dal cliente;
  • Ci permette di salvare dati su un database e riprenderli all'occorrenza;
  • PHP è dinamico, molto simile a Javascript ma con l'aggiunta di più indicizzazione nei motori di ricerca;
  • Con XAMPP, abbiamo un server in locale scrivendo come URL "localhost";
  • Poi vai su "http://localhost/phpmyadmin/";
  • Il database è un insieme di tabelle di dati;
  • Di solito, c'è un solo database per ogni app;
  • PHP è NATO per i database, dunque va di pari passo con MySQL;
  • RICORDA: Mai lasciare una pagina bianca su PHP ma usa SEMPRE il testo "" così da interpretare il codice per bene;
  • Per i commenti su PHP, usa i doppi trattini;
  • PHP ha variabili, cicli, condizioni, funzioni e oggetti;
